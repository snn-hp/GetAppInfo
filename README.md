上滑下滑 底部导航栏 + tablayout 隐藏显示联动

SlidingTabLayout2 支持ViewPager2,RecyclerView 循环滚动，无限滚动，卡片抽奖，卡片翻转

viewpager + tablayout 跨页面 Behavior联动效果
获取手机上已安装应用信息，系统应用和 非系统应用
MD5 SHA1 SHA256 签名信息 点击签名信息可复制到剪切板
新增搜索包名 appname
